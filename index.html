Validation Rules

35개 BMS 검증 항목 (검증 규칙)

GROUP A · 기본 상태 (항목 1–10)
unix_time — 수집 주기(3s~60s) · NULL 검사
ignit_status — 주기 · NULL · 값(0/1)
chg_conr_status_list — 충전 커넥터 상태
em_speed_kmh — 속도 범위 0~250 km/h
pack_curr / pack_volt — 전류·전압 범위
main_relay_status — 릴레이 주기·값
soc_display_rate / soc_rate — SOC 0~100%
mile_km — 주행거리 선형 증가
GROUP B · 셀 전압 (항목 11–16)
cell_min/max_volt — 범위 2.6~4.5V
cell_volt_dev — max − min 편차 계산
cell_min/max_volt_no — 번호 ≤ cell_count(N)
cell_voltage_1~N — 수집 윈도우 마스크 + 주기
GROUP C · 모듈 온도 (항목 17–20)
module_min/max_temp — 범위 −40~+85°C
module_avg_temp — 수집 윈도우 마스크
battery_module_1~N — 마스크 + ≤ module_count
GROUP D · 에너지·전력 (항목 21–27)
oper_second — 선형 증가 ≤1초/샘플
acc_chg/dchg_ah — 누적 충방전 전류량 ≥0
acc_chg/dchg_wh — 누적 충방전 전력량 ≥0
pack_pwr — pack_volt × pack_curr 교차검증
ir — 내부저항 ≥ 0
GROUP E · 복합 로직 (항목 28–35)
충전 감지 시간 — 충전 시작 기준 −5s~+25s
시작/종료 상태 — (ignit,chg,relay) 유효 튜플 검증
슬립 지연 시간 — IGN=0 이후 슬립까지 측정
soh_rate — 배터리 건강도 0~100%
IGN × 속도 — 주행 중 IGN 교차 검증
IGN × 릴레이 / 충전 × 릴레이 — 복합 상태 검증
충전 조건 — speed=0, relay=1, |curr|>1
그룹별 검증 항목 분포 (총 35개)

신규 기능 A · HTML Report

HTML 검증 리포트 생성

개요

기존 XLSX 리포트에 더해 HTML 형식 리포트를 함께 생성합니다.
브라우저에서 바로 열어 확인 가능하며, 이메일·공유 링크로 배포할 수 있습니다.

XLSX 대비 HTML 리포트 장점

✓
Excel 없이 브라우저만으로 열람 가능
✓
PASS/FAIL 색상 코딩 + 인터랙티브 필터
✓
요약 헤더(전체 통계) + 상세 테이블 일체화
✓
비엔지니어도 결과 이해하기 쉬운 레이아웃
